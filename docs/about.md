# 关于我

我关注计算机图形学、物理渲染、实时渲染、动画和物理模拟，希望理解公式背后的直觉，也重视它们最终如何落到工程实现中。

这个知识库用于长期整理学习过程中的推导、实现分析、课程笔记和参考资料。内容会持续校正与补充，而不是一次完成的静态归档。
