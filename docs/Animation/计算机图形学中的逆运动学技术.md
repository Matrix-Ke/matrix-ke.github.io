# 计算机图形学中的逆运动学技术调查 

**摘要：**
逆运动学 (IK) 是使用运动学方程来确定机械手的关节参数，从而使末端执行器移动到所需位置； IK 可以应用于许多领域，包括机器人、工程、计算机图形和视频游戏。在本次调查中，我们从计算机图形学的角度全面回顾了 IK 问题和多年来开发的解决方案。本文从正向和 IK 的定义、它们的数学公式开始，并解释了如何区分无法解决的情况，并指出何时有可用的解决方案。本报告中的 IK 文献分为四大类： 分析方法、 数值、 数据驱动 和 混合 方法。提供了说明关键方法的时间表，解释了 IK 方法多年来的进展情况。讨论了最流行的 IK 方法的性能、计算成本和结果姿势的平滑度，同时我们建议哪种 IK 系列求解器最适合特定问题。最后，我们指出了当前 IK 方法的局限性，并提出了未来的研究方向。 

## 1. 介绍
